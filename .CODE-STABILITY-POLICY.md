## Rules to Prevent Drift

**Rule 1: Source of Truth**
- Never duplicate core 10-code scripts (10_0XX_*.py)
- Canonical location: `hermes-config/10-codes/scripts/`
- Use them directly from that path

**Rule 2: No Bash Wrappers**
- Don't wrap Python scripts in bash unless it's a documented adapter
- Bash scripts are for ENV setup or composition, NOT reimplementation

**Rule 3: Use Existing Before Creating**
- ALWAYS search for existing implementation first:
  ```bash
  find $WORKSPACE/hermes-config -name "10_*" -type f
  ```
- Check memory/prior runs for context
- Ask before creating new "simpler" versions

**Rule 4: Integrity Checks**
- Run `.code-integrity-check.sh` before and after changes
- Must pass clean before committing
- Exit code 1 = violation detected

**Rule 5: Document Changes**
- Update this file if adding new canonical scripts
- Mark deprecated scripts with README files
- Link old paths to new locations with instructions
